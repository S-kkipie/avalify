# 04 — Naming decision

Final: **Avalify** — from *aval* (Spanish: guarantee / endorsement / backing) + *-ify* → "to make backed / vouched-for." On-thesis: Avalify gives its *aval* to a hire.

## Candidates evaluated (and why they died)

| Candidate | Verdict | Reason |
|-----------|---------|--------|
| **SentryLoop** | ✗ concept rejected | Monolithic gatekeeper framing; unbuildable custody phases; self-simulated demo trips anti-sybil. Pivoted to the broker concept. |
| **Vouch** | ✗ dead (same niche) | 4 collisions incl. `notifuturo/vouch` — a per-call payment trust/reputation API on x402 (near-identical to the pre-hire step) — plus two "Vouch Protocol" projects and "AgentVouch". |
| **Envoy** | ✗ dead (same niche) | `envoy-pay` = x402 + ERC-8004 agent-payment infra (exact niche); plus Envoy Proxy (CNCF) and Envoy visitor-management SaaS. |
| **Legate** | ✗ dead | Homophone **Ligate Labs** = attestation/reputation chain for AI (exact thesis); NVIDIA Legate; "Legate" Ruby AI-agent framework. |
| **Emissary** | ~ weak | Uniswap's *The Compact* uses `IEmissary` as an on-chain term — collision in the same EVM neighborhood. |
| **Attaboy / Bellboy** | ~ dropped | Pun route ("…boy", like en-*voy*). Low collision, but the pun angle was dropped. |
| **Tessera / Handsel / Kith** | ~ alternates | Coined/obscure, low collision, not chosen. |
| **Avalify** | ✓ chosen | **Zero collision in agent/web3/x402/crypto.** Meaning fits exactly. |

## Avalify collision notes

- **Clean in the niche that matters** — no agent/crypto project by this name.
- `avalify.com` is taken (a Brazilian iPhone-trade-in CRM) → use `avalify.xyz` / `avalify.ai` / `avalify.dev`.
- Out-of-niche homophones exist (Avify — LatAm WhatsApp CRM; Availify — app-name checker; Valiify — fintech; Evalify — patent-risk AI). None in agent/crypto → low real confusion for a hackathon agent.
- To confirm when claiming handles: npm, GitHub org, and social handles.

## Lesson

Every generic "messenger / delegate / vouch / trust" word is already taken in this hot space — several *in the exact niche*. A coined mark (Avalify) beats a dictionary word here.
